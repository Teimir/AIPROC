сommands
LDR - загрузить данные в регистр
ADD - сложить регистры
SUB - вычесть регистры
MOV - скопировать первый регистр во второй
POP - получить в регистр данные из стека
PUSH - положить в стек данные из регистра
JMP - безусловный переход по адресу / метке
JFZ - переход при регистр 1 равном 0 по адресу / метке
JFO - переход при регистр 1 равном 1 по адресу / метке
PRT - вывести значение регистра
HALT - остановка
CMP - сравнить регистры, результат в регистр флагов
GTF - получить во второй регистр значение из первого регистра
IN - положить значение во второй регистр из внешнюю шину по адресу из первого регистра
OUT - положить значение из первого регистра во внешнюю шину по адресу из второго регистра
Модуль озу работает по правилам
0 адрес - адрес в рам
1 адрес - данные 
2 адрес - 1 (чтение) 0 (запись)
3 адрес - 1 (активировать озу)
Соглашение по использованию шины
0-9 ОЗУ
10-19 ПЗУ
20-Конец остальное
